Academic Porject - DropBy

Worked from Apr. 2015 till Jul. 2016

Read Abstract

DropBy is developed during the course SEBA Master - Web Application Engineering at TUM in Summer Semester 2015. It is an online question-answer portal for the students of TUM inorder to help new students by taking advice from senior students. As new students have various queries from all walks of life, which could be easliy solved by the earlier residents in that area, i.e., their seniors. My task was to add the CRUD operations and user authentication in the same. Link to the repository. The project is created in Play framework using Java, Java Persistence API (JPA) and SQLite for the backend and Bootstrap, JavaScript, jQuery and AJAX for the frontend.
